## Description  

This project is based on the riwi HTML/CSS performance test. In this test, we will be creating a website about a rock band. This page will have several sections such as hero, events, about, and multimedia.

The page is composed of a header, which will contain the band's name and the navigation menu; the hero section, which will display a large image of the band; the about section, which will contain information about the band; and finally, the multimedia section, which will display some of the band's albums.

## Features  

### Navigation Menu  
- Hero
- Events
- about
- Multimedia

### Hero
Here you will see an image of the band.

### Events
This section will display a table of the band's upcoming concert dates.

### about
An image of the band will also be shown, and we'll talk a little more about them, their history, and how they started.

### Multimedia
In this section we will look at some of the band's most iconic albums
